Finding Classification
Finding Types

Every resource finding is classified by type, driving dashboard tabs, AI prompt selection, and savings calculations.

Cost (39 scanners) Security (7 scanners) Governance (4 scanners)
Severity Levels

Each finding carries a severity derived from its recommendation and type. Security findings use risk-based severity; cost findings scale by dollar impact.

Critical High Medium Low Info

Two AI prompt templates (cost-focused and security/governance-focused) are dispatched automatically based on the finding type mix in each scan.

Security — 4-Layer Read-Only Guardrails

Cloud Resource Sentinel cannot modify, delete, or create any AWS resources. This is enforced by four independent layers:

AI Prompt Guardrail
Advisory-only recommendations
SDK Interceptor
Blocks mutating API calls at runtime
Client Factory
All clients built with interceptor
Architecture Test
Build fails if factory bypassed
