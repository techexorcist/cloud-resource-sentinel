What Sentinel Finds

Cost waste, security gaps, and governance debt — surfaced in one scan

Cost & Idle
  • ● Stopped EC2 instances still costing money
  • ● Unattached EBS volumes and orphaned snapshots
  • ● Idle NAT Gateways, unused Elastic IPs
  • ● Oversized RDS, ElastiCache at <5% CPU
  • ● Empty S3 buckets, zero-traffic Lambda functions
  • ● Idle DynamoDB tables, empty SQS queues
Security & Governance
  • ● IAM users without MFA enabled
  • ● Unused IAM roles (no activity in 90+ days)
  • ● KMS keys without rotation enabled
  • ● Secrets Manager secrets without rotation
  • ● SSM parameters storing sensitive data in plain text
  • ● CloudTrail trails that aren't logging
AI Intelligence
  • ● Cross-resource correlation (EBS on stopped EC2)
  • ● Right-sizing recommendations with exact instance types
  • ● Phased cleanup plan (immediate, short-term, strategic)
  • ● Regional anomaly detection (isolated resources)
  • ● Well-Architected findings (what you're doing right)
  • ● Severity-ranked remediation for security findings
  • ● Executive summary with dollar amounts

How It Works

1
Connect

Pick an AWS profile (SSO supported), enter temporary credentials, or scan all accounts at once. Select regions and scan category.

2
Scan

50 resource types scanned across all selected regions simultaneously. CPU utilisation, traffic patterns, access history, and configuration are checked against idle thresholds.

3
Optimise

Review the interactive dashboard with charts and breakdowns. Enable AI for right-sizing, risk assessment, and a prioritised cleanup plan.

Features

Everything you need to manage cloud cost and hygiene

🧠
Multi-Model AI

Claude via AWS Bedrock or any local Ollama model (llama, mistral, gemma, qwen, deepseek, phi, and more). Switch models per scan. Small models auto-filtered for full scans to prevent timeouts.

👥
Multi-Account Batch

Scan all AWS profiles in one click. Scans queue automatically. SSO profiles auto-detected. Select regions by group (US, EU, APAC) or individually.

💰
Accurate Cost Estimates

Per-region pricing for all 30 AWS regions. Costs reflect your actual region — not generic US pricing. Refresh rates from AWS anytime. Reserved Instance and Savings Plan coverage detected to avoid false positives.

⚡
Fast Parallel Scanning

All 30 regions scanned simultaneously. A full account scan with 50 resource types completes in minutes, not hours. Real-time progress tracking across all pages.

🔗
Hidden Cost Detection

Finds waste that single-resource tools miss: EBS volumes on stopped EC2, Elastic IPs on idle instances, NAT Gateways in empty VPCs, orphaned snapshots for deleted databases.

🔄
Compare & Track

Up to 3 reports cached per account. Compare any two to see added, removed, and changed resources with cost, security, and governance finding deltas.

📊
16 Analytics Views

Dashboard with finding-type tabs, resources table with severity sorting, stats with finding-type and severity distribution charts, compare, audit trail, and cost/region/recommendation breakdowns.

📜
Audit Trail

Every scan logged with timestamp, account, status, resource count, cost, savings, AI provider, token usage, and duration.

🛡️
Security Scanning

IAM users without MFA, unused roles, KMS keys without rotation, secrets without rotation, sensitive params in plain text.

📤
Export & Share

Export scan results to CSV with costs, recommendations, and AI reasoning. Share findings with your team, attach to Jira tickets, or import into spreadsheets for further analysis.

🌐
Swagger API

Full OpenAPI 3.1 documentation with interactive Swagger UI. Test every endpoint directly from the browser.

🐳
Zero Setup

One command to start: docker-compose up or ./mvnw spring-boot:run. No database required. No configuration files to edit. Just point it at your AWS credentials and scan.

SAFETY GUARANTEE

Read-Only by Design

Sentinel can never modify, delete, or create AWS resources. This isn't a policy — it's enforced by four independent layers that make it architecturally impossible.
